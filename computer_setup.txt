Steps for setting up a new computer:

Linux mint:
    - Reverse mouse scrolling
    - Change time to standard time and add date. Monday first day of the week.
    - Don't allow grouping windows by application
    - Properly arrange monitor
			- wget -O /tmp/framework-laptop-tldr.sh https://raw.githubusercontent.com/lightrush/framework-laptop-formula/main/framework-laptop-tldr.sh && bash /tmp/framework-laptop-tldr.sh
    - Change default mouse pointer
    - Set background to black by right clicking and setting as wallpaper
			- Have to put black.png in user folder
    - Remove desktop icons
		- Change theme, folders and stuff to grey
		- Disable Desktop and Window Effects
		- Show battery percentage and time remaining. Right click and configure
    - Install Git
        - sudo apt install git
    - Install node.js
        - https://github.com/nodesource/distributions/blob/master/README.md
		- Install doctoc
				- sudo npm install -g doctoc
