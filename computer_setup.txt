Steps for setting up a new computer:

Linux mint:
    - Reverse mouse scrolling
    - Change time to standard time and add date. Monday first day of the week.
		- Add Panel to all monitors
			- System Settings -> Panel -> Add new panel
		- Panel settings
			- Add application to panels
			- Add application to panels and don't allow grouping windows by application
				- Right click on panel -> Applets -> (Click)Grouped Window List -> Setting button to the right -> uncheck Group windows by application
			- Decrease font size of panel to 28
			- No apps pinned to panel
				- Right click on apps on panel and unpin to panel
				- Just use window key to popup explorer
    - Properly arrange monitor
			- wget -O /tmp/framework-laptop-tldr.sh https://raw.githubusercontent.com/lightrush/framework-laptop-formula/main/framework-laptop-tldr.sh && bash /tmp/framework-laptop-tldr.sh
    - Change default mouse pointer
    - Set background to black by right clicking and setting as wallpaper
			- Have to put black.png in user folder
    - Remove desktop icons
		- Change theme, folders and stuff to grey
		- Disable Desktop and Window Effects
		- Show battery percentage and time remaining. Right click and configure
    - Install Git
        - sudo apt install git
    - Install node.js
        - https://github.com/nodesource/distributions/blob/master/README.md
		- Install doctoc
				- sudo npm install -g doctoc

- Start up applications
		- Command: gnome-terminal
    - Brave
		- VS Code

				- Edit -> Preferences
					- Set to fullscreen when clicked. Initial terminal size to max size.
					- Disable all shortcuts besides copy, paste, and zoom in/out
					- Change background and text color to black and white
					- Change built in scheme to linux console
					- Put CodeNewRoman fonts into ~/.local/share/fonts and change to that font in the terminal.

				- Run node installExtentions.js

					- au BufEnter * set fo-=c fo-=r fo-=o
				- Install Neovim Color Theme
					- If you need to redo the theme
						- https://themes.vscode.one
						- vsce package
					- Three dots then install from VSIX
		- Brave
			- Install using terminal from this website: https://brave.com/linux/
			- Settings
				- Uncheck: Show home button
				- Uncheck: Show Sidebar button
				- 80%: Page zoom
				- Uncheck: Show Brave Wallet icon on toolbar
				- Blank page: New tab page shows
				- Uncheck: Show Brave Rewards icon in address bar
				- DuckDuckGo: Normal Window - Search engine used in the address bar
				- DuckDuckGo: Private Window - Search engine used in the address bar
				- Uncheck: Offer to save passwords
				- Uncheck: Save and fill payment methods
				- Hamburger menu -> Show Sidebar -> Never
				- Check: Improve search suggestions
				- Check: memory saver
				- Dark: Brave colors
				- Go to a youtube video -> To the left of the address bar there's a button that's 2 circles with lines to their left and right -> Site settings -> Autoplay block

				- Video Speed Controller - video manager
					- Settings -> Preferred speed: 2.8

Firefox
- Settings
	- Open previous windows and tabs
	- Dark theme
	- Homepage and new windows/New tabs set to Blank Page
	- Default search engine set to Duckduckgo
	- Remove ask to save login and passwords
	- Remove Allow Firefox to install and run studies/Allow firefox to send technical and interaction data to Mozilla
	- Uncheck Control media via keyboard, headset, or virtual interface
- Default zoom: 80%
